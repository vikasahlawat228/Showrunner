---
description: Plan a complete story arc from concept to scene-level outlines
---

1. Gather the concept: ask the writer for the arc's theme, conflict, and resolution
2. Use the brainstorm_agent skill to generate 5-10 "what if" scenarios
3. Select the best direction with the writer
4. Use the story_architect skill to create the full hierarchy (Arc → Acts → Chapters → Scenes)
5. Create container buckets for each structural element
6. Use the research_agent skill for any real-world topics referenced
7. Commit the structure: `showrunner git stage-story && showrunner git commit-message`
