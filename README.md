# Antigravity

**Agentic AI Tool for Manga & Comic Creation**

Antigravity is a CLI tool that pairs with LLMs (like Gemini, Claude, GPT-4) to help you write complex, consistent narratives. It features an autonomous "Director" agent that can orchestrate the writing process for you.

## Quick Start

1.  **Install**:
    ```bash
    pip install .
    pip install litellm
    ```
2.  **Setup API Key**:
    Create `.env`: `GEMINI_API_KEY=your_key`
3.  **Run Director**:
    ```bash
    antigravity director action
    ```
